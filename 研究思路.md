# 研究思路整理

## 黄河三角洲特定要素的解译标志构建
1.解译标志可细分为自然岸线与人工岸线两大类（这个不清楚怎么划分）

2.确定海岸线到底是什么：用MNDWI辅助目视解译
（1）已经很淡的细长部分是否也需要画
（2）碎片化的地方如何确定
（3）有很长的河深入内陆怎么确定 

3.数据图两个问题：（1）右侧有拼接缝，真彩色下很淡，假彩色下很明显的色差
（2）最后裁剪后周围有无效黑边
（3）2024的大图无法正常生成MNDWI图

## 模型改进
- **主模型**: U-Net++和ResNet50(ImageNet)
- **引入注意力机制与多尺度感知**: 引入BAM注意力机制以抑制背景噪声；并利用ASPP模块应对地物的多尺度差异。
- **损失函数**: 引入 Sobel-Edge Loss，采用加权组合的方式构建总损失函数BCE+DICE+Sobel-Edge Loss
- **优化器**: AdamW
- **学习率**: 余弦退火算法Warmup（预热） + Cosine Annealing（余弦退火）
- **精度评估**：（1）IoU、Kappa 系数、F1 分数（2）均方根误差 (RMSE)：在提取出的岸线上随机选取验证点，逐点计算其到参考岸线的欧氏距离 。

## 训练调参
1.输入块：512×512，以及256×256（这个有点太小了，感觉会缺失宏观语义）

2.数据增强：旋转翻转（这里面代码是随机生成的，每一次看到的方向都不一样，这样既避免了数据量过大且保证了效果），继续增强

3.epoch：epoch早停机制，给他尽可能多的epoch，在终端中输入下面这些代码，得到最优解的epoch数
```javascript
python train.py --epochs 150 --warmup_epochs 5 --patience 15 --save_dir save
```

4.学习率lr:先预热再降低，避免刚开始跨度太大，中期太慢陷入局部，

5.Batch Size:（1）8：多出一张，每次是随机抽取哪个不要 ，保证精度

6.混合损失函数的权重配比：初始的是0.4BCE + 0.4Dice + 0.2*Edge
